# ⚡ FREELANCE_NODE ⚡

O **Cypher-Store** é uma plataforma de serviços (Cyber-Store) desenvolvida para atuar como o epicentro da minha presença digital como desenvolvedora. Ele subverte o conceito de portfólio tradicional, adotando uma estética **Brutalista** e **Cyberpunk** para oferecer uma experiência imersiva de "terminal de comando".

---

## 🛰️ CONCEITO & DESIGN

O projeto foi construído sob três pilares fundamentais:

1.  **Estética de Terminal (Low-Fi/High-Tech):** Uso de tipografia monoespaçada, alto contraste (Verde Neon e Vermelho sobre Fundo Preto) e elementos visuais que remetem a sistemas operacionais de segurança e hacking ético.
2.  **Identidade Regional (Fortaleza_Node):** O projeto orgulhosamente se posiciona como um nó da rede global operado em **Fortaleza, Ceará**, integrando a vibe tecnológica com a localização geográfica da desenvolvedora.
3.  **Conversão Direta (Business-Focused):** Diferente de um portfólio de estudos, o Cypher-Hub foca em **venda de serviços**. Cada "Node" na grade de serviços é um produto com preço base e CTA (Call to Action) para fechamento de contrato.

---

## 🛠️ ARQUITETURA TÉCNICA

O sistema utiliza uma abordagem **Vanilla-First** para garantir performance extrema e total controle sobre as animações:

### 🌑 CSS & Visual Engine
* **Scanner Line:** Uma animação linear infinita que simula o rastreamento de dados na tela.
* **Responsive Grid:** Sistema de malha flexível que se adapta de monitores Ultrawide a dispositivos móveis.
* **Glitch Effects:** Camadas de texto sobrepostas com deslocamento para criar a sensação de interferência digital.

### ⚙️ JavaScript Core (The Engine)
* **Smooth Navigation:** Sistema de rolagem suave para navegação entre módulos.
* **Lead Conversion:** Integração direta com a API do WhatsApp, gerando mensagens pré-configuradas baseadas no serviço selecionado pelo cliente.
* **Global Config:** Centralização de variáveis (Links, WhatsApp, IDs) em um único objeto para facilitar a manutenção.
